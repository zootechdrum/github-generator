# HTML TO PDF GENERATOR

## Description 
In this command-line application users can dynamically generates a PDF profile from a GitHub username. The application is invoked with the following command: node app.js .

### Technologies Used :

1. Node.js
2. EJS
3. Axios
4. GitHub API
5. html-pdf
6. Inquirer
7. Express
